How to?
This application compiles Saito modules into *.saito files that can be installed directly in a browser. Simply ZIP your module directory and drag-and-drop the file to this application.
More specific instructions on compiling applications files is available here on our wiki. If you are new to Saito please see our tutorials for help building applications.
If you run into problems after installation, you can uninstall the application or nuke your browser to remove any installed modules.
